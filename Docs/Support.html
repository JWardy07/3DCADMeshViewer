Supported formats

STL, OBJ, PLY, 3MF, STEP (.step/.stp)

Troubleshooting

1. If import fails, make sure the file extension is correct (.stl, .obj, .ply, .3mf, .step, .stp).

2. Try copying the file to On My iPhone / On My iPad and import again.

3. For STEP files, ensure the file is true geometry data, not a text clipping or shortcut file.

4. If the app behaves unexpectedly, close and reopen the app and retry.
